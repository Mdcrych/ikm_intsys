# 🏥 Прогноз медицинских страховок (IKM IntelSys)

Градиент-интерфейс для предсказания стоимости медицинской страховки в США  
на основе данных пациента. Модель — **Random Forest Regressor**, обученный  
на [Medical Cost Personal Dataset](https://www.kaggle.com/datasets/mirichoi0218/insurance).

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Положить insurance.csv в корень проекта
#    (скачать: https://www.kaggle.com/datasets/mirichoi0218/insurance)

# 3. Обучить модель
python train.py

# 4. Запустить веб-интерфейс
python app.py
```

## Структура проекта

```
├── app.py              # Gradio-интерфейс + функция предсказания
├── train.py            # Обучение, диагностика, сохранение модели
├── requirements.txt    # Зависимости Python
├── tests/
│   ├── test_app.py     # Тесты валидации входных данных и предсказания
│   └── test_train.py   # Тесты SimpleSmokerRule и логики масштабирования
└── README.md
```

## Переменные окружения

| Переменная | По умолчанию | Описание |
|---|---|---|
| `CSV_PATH` | `insurance.csv` | Путь к исходному датасету |
| `MODEL_PATH` | `ikm/best_insurance_model.pkl` | Путь к сохранённой модели |
| `MODEL_OUTPUT_PATH` | `best_insurance_model.pkl` | Путь для сохранения модели из train.py |
| `PLOT_OUTPUT_PATH` | `error_plot.png` | Путь для сохранения графика |

## Запуск тестов

```bash
pytest tests/ -v
```
